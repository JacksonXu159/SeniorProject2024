Dashboard:is an interface designed for personal investors to monitor and manage their investment portfolio.
There are multiple tabs, each providing different portfolio information.
The tabs are: dashboard, balances, holdings, activity, and portfolio watch.

The dashboard tab provides a high-level overview of the user's portfolio. It includes an accounts section detailing balance and rate of return.
There is a graph section showing performance over a specified date range, as well as a section showing the portfolio composition (the percentage weights of different asset classes).
There is a brief summary on recent transactions.
